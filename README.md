## Completed Tasks

### Task 1: Neomorphic 3D Portfolio
A high-performance personal portfolio site built using:
- **UI Core**: Semantic HTML5 & Vanilla CSS3
- **3D Graphics**: Three.js WebGL rendering (ambient rotating low-poly Icosahedron background)
- **Animations**: GSAP & ScrollTrigger (scroll scrubbing rotation, GPU-composite section fading)
- **Design Language**: Tactile Red-and-White Neomorphism (raised vs. inset shadows, Outfit & Inter typography)
- **Intro Experience**: Text expansion animation emerging from a central crimson shine with preloader fade-out.

#### Running the Portfolio Locally
1. Navigate to the `/portfolio/` subdirectory:
   ```bash
   cd portfolio
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Access the site in your browser at `http://localhost:3050`.

*Note: On Windows, you can double-click [`Launch_Vibhath_3D_Portfolio.bat`](./portfolio/Launch_Vibhath_3D_Portfolio.bat) inside the `portfolio` directory to run the server and open the browser automatically in one click.*
